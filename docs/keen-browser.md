# Keen Browser

**Browse Untamed.**

Keen Browser is a fork of Brave strictly for Android TVs.

It adds remote-friendly pointer navigation, edge scrolling, keyboard-aware
input, smooth pointer movement, automatic pointer fading, Android TV launcher
support, and Keen branding. Brave's privacy protection and ad-blocking engine
remain intact.

Keen has been tested on a Mi Box with D-pad navigation, clicking, scrolling,
search submission, text editing, and Android TV home-screen launching.
